Dependencies:
MATLAB                                                Version 9.10        (R2021a)
Deep Learning HDL Toolbox                             Version 1.1         (R2021a)
Deep Learning Toolbox                                 Version 14.2        (R2021a)
Image Processing Toolbox                              Version 11.3        (R2021a)
Deep Learning Toolbox Converter for TensorFlow Models ***

***This final tool box must be installed manually. If not installed the code will generate an error and you will not be propmted to install the toolbox.

HOW TO USE:
1. Open the 'run' file
2. Enter the image or images to be classified in 'image_filepath'
3. The classification will be output into 'I', with 1 pixels representing lymphoid tissue and 0 pixels representing non lymphoid tissue.
